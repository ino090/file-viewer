ブラウザだけで動くファイルビューア集

Excel・CSV・JSON・Word・ZIPファイルを、インストールや会員登録なしでその場で閲覧できます。 すべての処理はお使いのブラウザ内で行われ、ファイルはどこにもアップロードされません。

📊

Excel ビューア

シートを切り替えながら、表形式でセル内容を確認できます。

.xlsx / .xls / .xlsm / .csv
📋

CSV ビューア

区切り文字・文字エンコーディングを指定して表形式で表示します。

.csv / .tsv / .txt
🧩

JSON ビューア

整形表示とツリー表示で、入れ子になったデータを見やすく確認できます。

.json
📄

Word ビューア

Word文書(.docx)をHTMLに変換し、見た目に近い形で表示します。

.docx
🗜️

ZIP ビューア

ZIP内のファイル一覧を表示し、テキストファイルの内容を確認・取り出しできます。

.zip
🔒 サーバー不要・送信なし ファイルは外部サーバーへ送信されず、すべてブラウザ内（JavaScript）で処理されます。
🖱️ ドラッグ&ドロップ対応 各ツールのページにファイルをドラッグ&ドロップ、またはクリックして選択できます。
📱 スマホ対応 スマートフォン・タブレットでも利用できるレスポンシブデザインです。
🚫 画像ビューアは非対応 画像ファイル（jpg, png, gif など）の表示には対応していません。文書・ZIP内の画像も表示されません。
ご利用にあたって
  • すべての処理はブラウザのメモリ上で行われます。表示後にページを閉じればデータは破棄されます。
  • 大きなファイルを開くと、表示までに時間がかかったり、ブラウザの動作が重くなる場合があります。
  • 古い形式のファイル（.doc, .xls など）は一部のみ対応、または非対応の場合があります。各ツールページの説明をご確認ください。
  • 画像ファイル単体の閲覧、および文書・ZIP内に含まれる画像の表示には対応していません。

ファイルビューア集 — すべての処理はブラウザ内で完結します（サーバー送信なし）

Excel: SheetJS / CSV: PapaParse / Word: mammoth.js / ZIP: JSZip & marked.js
