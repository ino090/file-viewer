🧩 JSON ビューア

JSONファイルをブラウザに読み込み、折りたたみ可能なツリー表示、または整形済みテキストで表示します。

  • ファイルはサーバーに送信されず、ブラウザ内のみで処理されます。
  • 非常に大きなJSONファイルは、表示までに時間がかかったりブラウザの動作が重くなる場合があります。
